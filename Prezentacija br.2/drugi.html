Promenljive


   Promenljive koristimo da ne bismo istu vrednost pisali više puta,
kao i da bismo mogli da pratimo o kom objektu se radi čak i kada mu se vrednost promeni.

Razlikujemo globalne i lokalne promenljive.
Globalne promenljive su dostupne u celom programu,dok su lokalne promenljive definisane unutar funkcije i neće biti dostupne van funkcije.

Primer globalne promenljive:                                 Primer lokalne promenljive:

var tekst = "Zdravo";

function ispis(){

alert(tekst);

}
ispis();
function sabiranje(){

var a =10;
var b = 30;

return a+b;
}
console.log(sabiranje());



JSON


   JSON(JavaScript Object Notation) je format koji dosta podseća na javascript objekte.
Javascript objekte možemo konvertovati u JSON i možemo ih konvertovati nazad iz JSON-a u objekat.

Primer Javascript objekta:                                      Isti primer konvertovan u JSON:

var tipoviPodataka = {
a : 7,
b : true,
c : {a : "b"},
d : "String",
e : [1,2,3]
};
{
"a" : 7,
"b" : true,
"c" : {
"a" : "b"
},
"d" : "String",
"e" : [1,2,3]
}

Razlika između JSON i Javascript objekta je što kod JSON-a imamo navodnike oko ključeva.

var jsonString3 = JSON.stringify(tipoviPodataka);

console.log(jsonString3);
 // Konvertovanje iz Javascript objekta u JSON.


Math. objekat


   Math.objekat nudi mnoge metode koje možemo da koristimo pri izračunavanju različitih matematičkih operacija.
Neke od metoda Math.objekata su:

  • Math.abs
  • Math.pow
  • Math.round
  • Math.sqrt
  • Math.ceil
  • Math.floor
  • Math.min
  • Math.max
  • Math.random
  • Math.PI

Primeri:

function obimKruga(r) {

return 2 * r * Math.PI;

}
obimKruga(); // Math.PI vraća broj PI pomoću kojeg izračunavamo obim kruga


function randomBroj(x, y) {

return Math.floor(Math.random() * (y - x)) + x;

}
randomBroj(); // Vraća random broj imeđu x i y
// Math.floor zaokružuje broj na dole, Math.random vraća random broj od 0 do 1


console.log(Math.ceil(6.4));
//Math.ceil zaokružuje broj na gore,tj.vraća prvi veći ili jednak celi broj


function razlika(a, b) {

return Math.abs(a - b);

}
console.log(razlika(2, 6));
// Math.abs vraća apsolutnu vrednost nekog broja.
// Negativan broj pretvara u pozitivan,a pozitivne brojeve ne menja.


function duzinaHipotenuzePravouglogTrougla(a, b) {

return Math.sqrt((a * a) + (b * b));

}
console.log(duzinaHipotenuzePravouglogTrougla(5, 8));
// Math.sqrt računa kvadratni koren nekog broja.


document.write(Math.pow(5, 3)); // Math.pow stepenuje broj
// Ima dva parametra,drugi parametar kaže za koji stepen treba povećati prvi parametar.


console.log(Math.max(2,28,24,15)); // Od preosleđenih brojeva vraća najveći.

console.log(Math.min(2,28,24,15)); // Od prosleđenih brojeva vraća najmanji.
