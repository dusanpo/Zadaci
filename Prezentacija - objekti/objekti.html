Objekti


   Objekti se kao i nizovi mogu koristiti za smeštanje više vrednosti u jednu promenljivu. Objekat je skup svojstava od kojih svako ima svoje ime(ključ) i vrednost.Pored svojstava u elemente objekta spadaju i metode. Metode su u stvari funkcije koje se nalaze u objektu.

var osoba = {
ime : "Bojan", // "ime" je ključ, "Bojan" je vrednost tog ključa
prezime : "Žižić",
zanimanje : "lekar"
};



Kreiranje objekta


Objekte možemo kreirati na više načina:

1.Object literal

Ovaj tip kreiranja objekata je najjednostavniji, i zasniva se na definisanju elemenata objekata izmedju vitičastih zagrada.

Primer:

var osoba = {
ime: "Milan",
godine: 35,
zanimanje: "poštar",
pozdrav: function () { // ovaj red možemo ispisati po skraćenoj sintaksi: pozdrav(){
return "Zdravo ja sam " + osoba.ime + ", imam " + osoba.godine + " godina.";
}
};

console.log(osoba); // funkciju pozivamo ovako: osoba.pozdrav();

Funkcije koje se nalaze u objektu zovemo metodi. Iznad kreirani metod "pozdrav" u objektu "osoba" pozvaćemo ovako:
osoba.pozdrav();
Elementima objekta pristupamo na jedan od ova dva načina:
osoba.ime;
osoba['ime'];
Ponekad ključevi moraju biti pod navodnicima,to se dešava kada u imenu ključa postoji neki "nedozvoljeni" karakter koji bi mogao pogrešno da se interpretira (npr.razmak,dve tačke), u tom slučaju pozivamo elemente objekta na drugi način(sa uglastim zagradama).
Ključevima možemo naknadno dodeljivati nove vrednosti:
osoba.godine = 40;
Svojstvo objekta možemo obrisati, to radimo pomoću delete. Na primer:
delete osoba.godine;

Ako želimo da objektu dodamo novo svostvo,to radimo na sledeći način:
osoba.status = "oženjen";(ime objekat.ključ = vrednost;)
Ako bi želeli da objektu dodamo nov metod,to radimo ovako:
osoba.brak = function () {...};

U sledećem primeru smo objektu "osoba",dodali novo svojstvo i novi metod:

var osoba = {
ime: "Milan",
godine: 35,
zanimanje: "poštar",
pozdrav: function () {
return "Zdravo ja sam " + osoba.ime + ", imam " + osoba.godine + " godina.";
}
};

osoba.brak = function () {
return "Moje zanimanje je " + osoba.zanimanje + " i " + osoba.status + " sam.";
};

osoba.status = "oženjen"; // dodajemo novi elemenat u objekat

console.log(osoba);



2.New Object

Ovde kreiramo instancu objekta sa new Object. Prvo kreiramo prazan objekat, a zatim pojedinačno dodajemo nove ključeve i vrednosti.

Primer:

var osoba1 = new Object(); // ovaj red može i ovako: var osoba1 = {};
osoba1.ime = "Dragan";
osoba1.prezime = "Jovanović";
osoba1.zanimanje = "student";
osoba1.zdravo = function () {
return "Ja se zovem " + this.ime + " " + this.prezime;
};

console.log(osoba1);
console.log(osoba1.zdravo());

Pomoću new Object() kreiramo nov,prazan objekat. Da bi metod pristupio ključevima objekta koristimo this.Reč this upućuje na trenutni objekat.


3.Object constructor(prototype)

Kreiramo objekat primenom konstruktorske funkcije.U samoj funkciji je jako značajna upotreba this koji predstavlja objekat za koji je funkcija pozvana.Svako svojstvo koje želimo da kreiramo definišemo kao this.svojstvo.
Koristimo rezervisanu reč new da bismo kreirali nov objekat na osnovu konstruktor funkcije.
Preporučljivo je da metode koje pravimo ne budu unutar funkcije već da ih izdvojimo.U tu svrhu koristimo prototype, koji se korisi za dodavanje metoda na konstruktor funkciju.

Primer konstruktor funkcije sa parametrima:

var Auto = function (marka, model, godProizvodnje) {
// ime konstruktora treba da počinje velikim slovom
this.marka = marka;
this.model = model;
this.godProizvodnje = godProizvodnje;
this.maxBrzina = 240;
};
Auto.prototype.prikaziAuto = function () {
return `Auto je: ${this.marka} ${this.model}, proizveden ${this.godProizvodnje}. godine.`
};
Auto.prototype.dajBrzinu = function () {
return this.maxBrzina;
};

var mojAuto = new Auto("Opel", "Astra", 2014); // ovde smo kreirali objekat

console.log(mojAuto.prikaziAuto());
console.log("maksimalna brzina:", mojAuto.dajBrzinu(), "km/h.");

imeKonstruktora.prototype.metod = function() {}; - ovo je šablon kada se koristi prototype.
Kao što smo u prototipu definisali metode isto tako smo mogli da definišemo i svojstva
(npr. Auto.prototype.marka = "Opel";). Međutim ovo bi trebalo da se izbegava, jer ako promenimo vrednost nekog svojstva ta promena se neće desiti u prototipu, već će se kreirati novo svojstvo samog objekta.

var mojAuto = new Auto("Opel", "Astra", 2014); - kreirali smo promenljivu i rekli da je ona jednako new Auto, takodje smo i definisali parametre.Na ovaj način smo kreirali objekat i pozvali funkciju.

Primer konstruktor funkcije bez parametra:

function Car() { // može i ovako: var Car = function() {
this.manufacturer = "Citroen";
this.model = "C5";
this.productionYear = 2012;
this.speed = 80;
this.maxSpeed = 220;
};

Car.prototype.getSpeed = function () {
return this.speed;
};

var car3 = new Car();

console.group("CAR3");
console.log(car3.getSpeed());
console.log(car3);



Module Pattern

Modul Patern je vraćanje promenljivih,funkcija i celih objekata iz IIFE(Immediately Invoked Function Expression). IIFE je funkcija koju možemo da izvršimo čim je kreiramo. Kreira se tako što funkciju obavijemo zagradama i na kraju dodamo još jedan par zagrada kako bi mogli odmah da je pozovemo:

(function () {

})(); // zadnji par zagrada služi da se pozove funkcija i da se daju parametri za pozivanje

Opseg važenja onoga što upišemo u IIFE ostaje unutar funkcije,nije javno dostupno i neće moći da bude pozvano van nje. Sav kod unutar ovakve funkcije ostaje privatan.

Primer:

(function () {
var sati = new Date().getHours();
var pozdrav;
if (sati < 18) {
pozdrav = "Dobar dan";
} else {
pozdrav = "Dobro veče";
}
console.log(pozdrav); // ovde je u redu
})();

console.log(pozdrav); // vraća : error , pozdrav is not defined

Možemo da izaberemo koje podatke želimo da zadržimo kao privatne a koje podatke želimo da objavimo. Ono što hoćemo da bude javno dostupno nakon pozivanja programa stavljamo u return. Na ovaj način možemo učiniti dostupnim sve tipove podataka.

Primer vraćanja objekta:

var izracunajProsek = (function (nizBrojeva) {
var brojevi = nizBrojeva;
var objekat = {};
objekat.prosek = function () {
var ukupno = brojevi.reduce(function (rezultat, element) {
return rezultat + element;
});
return "Prosek je: " + ukupno / brojevi.length;
};
return objekat; // ovde smo kreirali objekat
})([24, 152, 325, 18, 5265, 255]);

console.log(izracunajProsek.prosek());

Preko return smo praktično kreirali novi objekat.
U sledećem primeru ćemo videti kako možemo da izaberemo da neke metode ostanu privatne, a neke da budu javne i dostupne svima:

var mojModul = (function (nizOcena) {

var ocene = nizOcena; // JAVNA PROMENLJIVA

function vratiBroj() { // PRIVATNA METODA
var manji = ocene.filter(function (broj) {
return broj < 10;
});
return "Vrati brojeve manje od deset: " + manji.length;
};

function poredjaj() { // JAVNA METODA
var ukupno = ocene.sort(function (a, b) {
return b - a;
});
return "Poredjaj od najvećeg broja: " + ukupno;
};

return { // BIRAMO ONO ŠTO ĆE BITI JAVNO
ocene : ocene, // preko ključeva objekta odlučujemo šta želimo da bude javno
poredjaj : poredjaj, // u drugoj koloni "ocene" je promenljiva a "poredjaj" je metod
};
})([10, 5, 6, 9, 10, 7]);

console.log(mojModul.ocene); // [10,5,6,9,10,7]
console.log(mojModul.poredjaj()); // Poredjaj od najvećeg broja: 10,10,9,7,6,5
console.log(mojModul.vratiBroj()); // Error: vratiBroj is not a function



For/In petlja

For/in petlja prolazi kroz sva svojstva objekta. Ciklus se izvršava po jednom za svaki elemenat objekta. Vrednostima svojstva pristupamo koristeći uglaste zagrade.

Sintaksa:

for (promenljiva in objekat){
naredba
}

Primer:

let tekst = "";
let student = {
ime: "Petar",
fakultet: "FON",
prosekOcena: 8.6
};

let x;
for(x in student){
tekst += student[x] + "\n";
}

console.log(tekst);
