Nizovi

   Niz je struktura koju koristimo da sačuvamo više vrednosti u jednoj promenljivoj. Nizovima se pristupa preko rednog broja odnosno "indeksa". U Javascriptu prvi elemenat niza uvek je sa indeksom 0,drugi je sa indeksom 1,treći sa indeksom 2 itd.
Svakom elementu niza pristupamo preko njegovog indeksa koji navodimo u uglastim zagradama
(a[1] - elemenat niza a sa indeksom 1,niz ima dva elementa),
(a[1] = 120; - novu vrednost unosimo pod bilo kojim indeksom).


-Kreiranje nizova:

var a = []; // prazan niz(ima 0 elemenata)
var brojevi = [25,14,75,4,36]; // niz koji ima 5 elemenata
var imena = ["Ivana","Marko","Dušan","Sanja"]; // niz stringova,niz koji ima 4 elemenata
var miks = [21,true,"Milica"]; // niz koji sadrži različite tipove podataka

Ako su elementi niza nizovi onda se radi o multidimenzionalnim nizovima. Ovde prvo moramo da kreiramo niz, pa onda da za svaki elemenat tog glavnog niza kreiramo podniz.
Kasnije u konzoli možemo pozvati koji god elemenat niza želimo.

var multi = [[8,12,24],[7,3,14],[26,18,45]]; // primer multidimenzionalnog niza
console.log(multi[1]); // vraća drugi niz [7,3,14]
console.log(multi[2][1]); // iz 3.niza vraća 2.elemenat(vraća broj 18 iz niza[26,18,45])


-Prolazak kroz elemente niza

Kroz elemente niza ćemo najlakše proći pomoću FOR petlje:

var automobili = ["Mercedes", "Opel", "Fiat"]; // niz sa tri elementa (indeksi 0, 1 i 2)
for (var i = 0; i < 3; i++) {
console.log(automobili[i]); // u konzoli ispisuje jedan po jedan elemenat niza(Mercedes,Opel,Fiat)
}

Primer prolazka kroz elemente multidimenzionalnog niza:

var m = [[28, 22], [71, 11], [23, 81]]; // svaki od tri niza ima po dva elemenata
for (var k = 0; k < 3; k++) {
for (var j = 0; j < 2; j++) {
console.log(m[k][j]); // u konzoli ispisuje jedan po jedan elemenat niza(28,22,71,11,23,81)
}
}


Metode za rad sa nizovima:


.length

Svi nizovi imaju svostvo length koje predstavlja broj elemenata u nizu. Ovo svojstvo uglavnom koristimo za čitanje,kako bismo znali koliko niz ima elemenata. Koristimo ga ako želimo da postavimo novu veličinu niza,tako njegovim smanjenjem možemo da skratimo niz.
Važno je zapamtiti da length predstavlja broj elemenata niza a ne indeks poslednjeg elementa. To znači da je indeks poslednjeg elementa uvek za jedan manji od vrednosti svojstva length.

Primeri:

var sportovi = ["fudbal", "košarka", "tenis", "rukomet", "odbojka"];
for (var g = 0; g < sportovi.length; g++) {
console.log(sportovi[g]);
} // nabrajamo sve elemente niza pomoću petlje

var b = [100, 200, 300, 400, 500];
for (var i = 0; i < b.length; i++){
console.log(i + " : " + b[i]);
} // u konzoli se ispisuje indeks i vrednost elemenata niza odvojeni sa dve tačke


.push()

Metod push() koristimo za dodavanje jednog ili više elemenata na kraj niza.

Primer:

var namestaj = ["sto", "krevet" , "orman"];
namestaj.push("televizor");
console.log(namestaj);
// ova metoda kao rezultat vraća novu dužinu niza
// dodat je novi elemenat na kraju niza:["sto", "krevet" , "orman", "televizor"]


.unshift()

Pomoću metoda unshift() dodajemo jedan ili više elemenata na početak niza. Time će indeksi postojećih elemenata biti promenjeni.

Primer:

var brojevi = [12,19,34,56];
brojevi.unshift(2,8); // dodata su dva nova elementa na početku niza
console.log(brojevi); // kao rezultat dobijamo novu dužinu niza:[2,8,12,19,34,56]


.pop()

Metod pop() uklanja poslednji elemenat iz niza,zatim kao rezultat vraća taj isti elemenat.

Primeri:

var voce = ["banana", "limun", "jabuka" , "kivi", "breskva"];
console.log(voce.pop()); // ovde kao rezultat vraća zadnji elemenat niza.


var vocke = ["banana", "limun", "jabuka" , "kivi", "breskva"];
vocke.pop();
console.log(vocke);
// u ovom slučaju vraća sve elemente niza osim poslednjeg:["banana","limun","jabuka","kivi"]


.shift()

Metod shift() uklanja prvi elemenat iz niza,zatim kao rezultat vraća uklonjeni elemenat.Dužina niza se smanjuje za jedan.

Primer:

var niz = [50,100,200,400];
var prviElemenat = niz.shift();
console.log(niz); // vraća niz bez prvog elementa:[100,200,400]
console.log(prviElemenat); // vraća prvi elemenat niza


.splice()

Metod splice() nam omogućava da istovremene dodajemo i uklanjamo elemente na bilo kom mestu u nizu. Može se zadati u obliku sa dva ili više parametra:
- Prvi parametar je pozicija na koju dodajemo,to je indeks elementa od koga počinje dodavanje. Ako se zada indeks veći od poslednjeg elementa,podrazumeva se dužina niza tj.prvi indeks posle poslednjeg elementa. Ako se zada negativan broj,odbrojava se toliko elemenata od kraja niza.
- Drugi parametar označava broj elemenata koji se uklanjaju, počev od elementa čiji je indeks zadat. Ako se zada broj 0, onda se ništa ne uklanja iz niza.Ako se zada veći broj nego što ima elemenata do kraja niza, onda se uklanjaju svi elementi do kraja.
- Svi parametri posle toga su novi elementi koje hoćemo da dodamo.Ako smo uklanjali neke elemente iz niza,novi elementi se ubacuju umesto njih. Ako nismo uklanjali elemente, novi elementi se ubacuju ispred elementa čiji smo indeks zadali.

Primeri:

var godine = [2012,2015,2016,2017,2018];
godine.splice(1,0,2013,2014); // ispred indeksa 1 dodajemo nove elemente
console.log(godine); // u konzoli se ispisuje: [2012,2013,2014,2015,2016,2017,2018]


var meseci = ["Januar","Maj","Mart","April"];
meseci.splice(-3,1,"Februar"); // uklanjamo treći elemenat od kraja niza i umesto njega dodajemo novi elemenat
console.log(meseci); // u konzoli se ispisuje: ["Januar","Februar","Mart","April"]


.indexOf()

Ovaj metod vraća prvo pojavljivanje elementa u nizu.Kao rezultat vraća indeks pronađenog elementa. Ako se zadata vrednost ne pronađe rezultat je -1.
Metod indexOF() ima dva parametra.Prvi parametar je vrednost koja se traži i taj je obavezan. Drugi parametar je broj indeksa od koga počinje pretraga,ako se ne navede podrazumeva se početak niza tj.indeks 0. Drugi parametar može biti i negativan.U tom slučaju se elemenat od koga počinje pretraga odbrojava od kraja niza.

Primeri:

var motori = ["Yamaha","Honda","Suzuki","Kawasaki"];
console.log(motori.indexOf("Suzuki"));
// u konzoli ispisuje 2,jer je traženi elemenat sa indeksom 2


var num = [22,33,44,55,66,22,77];
var x = num.indexOf(22, 2); // tražimo broj 22 i krećemo pretragu od indeksa 2(treće pozicije)
console.log(x); vraća 5(preskače prvu 22 i nalazi sledeću koja je na indeksu 5)

/* var num = [22,33,44,55,66,22,77];
var x = num.indexOf(22, -3);
console.log(x);
ako je drugi parametar negativan kao u ovom slučaju -3,tada krećemo pretragu od 3.elementa od kraja */

U sledećem primeru vidimo kako da pronađemo drugu pojavu nekog elementa:

var n = [10, 20, 35, 40, 20, 55, 60]
var p = n.indexOf(20);
if (p > -1) {
p = n.indexOf(20, p + 1);
}
console.log(p); // vraća 4(drugu 20-icu u nizu)
/* ovde smo pronašli prvi indeks tražene vrednosti i kad je element ponađen ,ponovo ga tražimo,ali počev od prve sledeće pozicije */

.join()

Metod join() služi da sve elemente niza pretvori u string,tako što između svakog elementa ubaci ,(zarez). Podrazumevani separator je ",". Ako se zada prazan string onda će svi elementi biti spojeni.

Primeri:

var br = [[1, 2, 3], [4, 5, 6], [7, 8, 9]];
var y;
y = br.join(" - ");
console.log(y); // vraća 1,2,3 - 4,5,6 - 7,8,9


var s = ["lekar", "pekar", "apotekar"];
var t;
t = "zanimanje:" + s.join(" zanimanje:");
console.log(t); // zanimanje:lekar zanimanje:pekar zanimanje:apotekar


.toString()

Ovaj metod vraća prikaz niza kao string.Elementi su odvojeni zarezima.

Primer:

var zivotinje = ["tigar","medved","lav","zec"];
console.log(zivotinje.toString()); // tigar,medved,lav,zec


.slice()

Metod slice() vraća podniz od datog niza. Dva parametra određuju početak i kraj podniza koji se dobija. Rezultat je niz koji sadrži elemenat određen prvim parametrom(zapamtili smo da brojanje kreće od nule) i broji sve naredne elemente do drugog parametra(ali ne i njega).
Ako je naveden samo jedan parametar,rezultat sadrži sve elemente počev od tog parametra pa sve do kraja niza.

Primeri:

var dani = ["ponedeljak", "utorak", "sreda", "cetvrtak", "petak", "subota", "nedelja"];
var vikend = dani.slice(5,7); // var vikend = dani.slice(5); - dobija se potpuno isti rezultat
console.log(vikend); // vraća: ["subota", "nedelja"]

Ako su kao u sledećem primeru,parametri negativni,onda se broji od kraja niza(-1,-2,-3...).

var polugodiste = ["Januar", "Februar", "Mart", "April", "Maj", "Jun"];
var prolece = polugodiste.slice(-4,-1);
console.log(prolece); // ovde vraća: ["Mart", "April", "Maj"]


.sort()

Ovaj metod, ako ga zadamo bez ikakvih parametara, sortira elemente niza u rastućem poretku(od najmanjeg ka najvećem). Važno je napomenuti da se u ovom slučaju svi elementi niza ponašaju kao stringovi.Bez obzira o kom tipu podataka se radi,svi elementi niza će biti poređani u zavisnosti od njihove pozicije u UNICODE tabeli("100" ide pre "3",isto kao što je "aa" ispred "c").

unicode-table.com

Primeri:

var names = ["Mitar", "Anabela", "Ana", "Slavko", "Dragan"];
names.sort();
console.log(names); // vraća: ["Ana", "Anabela", "Dragan", "Mitar", "Slavko"]


var numbers = [1, 7, 100, 238, 10, 33, 21];
numbers.sort();
console.log(numbers); // vraća: [1, 10, 100, 21, 238, 33, 7]


Metod sort() je mnogo korisniji ako ima parametar koji predstavlja funkciju, koja se poziva pri svakom upoređivanju.Sada možemo da napravimo bilo koje svoje pravilo za sortiranje. Funkcija ima dva parametra koji predstavljaju dva elementa niza koji se u tom trenutku uporedjuju.

var niz1 = [50, 3, 67, 128, 110, 25, 245, 84, 65, 9];
function poredjaj() {
niz1.sort(function (a, b) {
return b - a; // ako je a manje od b,vraća pozitivan broj
});
document.getElementById("ispis").innerHTML = niz1;
}

.concat()

Metod concat() spaja dva ili više niza.Kao rezultat dobijamo nov niz.

var boje1 = ["plava", "bela", "crvena"];
var boje2 = ["crna", "zelena", "siva"];

.every()

Metod every() služi da proverimo ispravnost svih elemenata niza.Vraća vrednost true ako svaki elemenat niza ispunjava uslov, u suprotnom vraća false.
Važno je napomenuti da ovaj metod vraća true ako funkcija vrati true za svaki elemenat niza. Ako funkcija u nekom trenutku vrati false nema potrebe da se ispituju ostali elementi niza, jer će ukupni rezultat biti false.

Primer:

var gdn = [32, 21, 18, 20, 65, 38];
function proveriGodine(){
var svi= gdn.every(function(element) {
return element >= 18;
});
document.getElementById("pregled").innerHTML = svi;
} // proveravamo da li je svaki broj veći ili jednako 18




.some()

Ovaj metod će vratiti vrednost true ako makar jedan elemenat niza ispunjava uslov. Ako funkcija vrati true za bar jedan elemenat u nizu,onda se ostali elementi niza i ne proveravaju.

Primer:

var poeni = [68, 92, 89, 34, 88, 82];
function maxPoena(){
var sto = poeni.some(function(elementi) {
return elementi == 100;
});
document.getElementById("maksimum").innerHTML = sto;
} // proveravamo da li u nizu postoji broj 100




.filter()

Metod proverava za svaki elemenat da li ispunjava uslov i formira novi niz sa onim elementima koji prolaze filter.

Primer:

var niz3 = [4, 6, 25, 8, 11, 32, 5];
function manjiOdDeset() {
var d = niz3.filter(function (broj) {
return broj < 10;
});
document.getElementById("pronadji").innerHTML = d;
} // vraća brojeve manje od 10




.reduce()

Pomoću ovog metoda ceo niz se pretvara u jednu vrednost.Za svaki elemenat niza, metod reduce() poziva funkciju i prosledjuje joj ukupni dotadašnji rezultat i sam elemenat.

Primer:

var niz4 = [10, 18, 9, 3, 41, 11];
function zbir() {
var q = niz4.reduce(function (rezultat,elemenat) {
return rezultat + elemenat;
});
document.getElementById("sum").innerHTML = q;
} // sabira sve elemente niza


Ovde promenljiva "rezultat" po defaultu kreće od nule.Sa svakim pozivom predstavljaće prethodni rezultat koji se prosleđuje funkciji zajedno sa sledećim elementom koji je na redu.Funkcija ih sabira i vraća kao novi rezultat koji će se proslediti u sledećem pozivu,sve dok ne postane krajnji rezultat.


.map()

Metod map() za svaki elemenat niza poziva funkciju i menja sve elemente niza. Kreira se novi niz u kome svaki elemenat odgovara jednom elementu iz originalnog niza.

Primeri:

var niz5 = [10, 5, 11, 2];
var stepenuj = niz5.map(function (broj){ // funkcija menja svaki elemenat iz niza
return Math.pow(broj, 3); // svaki broj iz niza se stepenuje
});
console.log(stepenuj); // [1000, 125, 1331, 8]



var rezultatiTesta = [51, 22, 71, 34, 47,89]; // broj poena na testiranju(max. je 100)
function paoIliPolozio(){
var ocene = rezultatiTesta.map(function(elem) {
var d = Math.ceil(elem / 10); // broj poena se deli sa 10 pa se zaokružuje
if(d < 6){ // ako ima manje od 51.poena, ocena je 5,nije položio
return " pao";
}
else{
return " položio";
}
});
document.getElementById("oceni").innerHTML = ocene;
}




.forEach()

Primer:

var omiljenaJela = ["pečenje", "riba", "roštilj", "špagete"];

.find()

Metod find() proverava sve elemente niza i vraća prvi koji ispunjava uslov funkcije. Ukoliko ni jedan elemenat ne ispunjava uslov,vraća undefined.

Primer:

var niz6 = [7, 22, 3, 15, 1];
function nadjiBroj() {
var pronadji = niz6.find(function (br) {
return br < (Math.sqrt(16));
});
document.getElementById("nadji").innerHTML = pronadji;
} // vraća prvi broj koji je manji od 4




Array.isArray()

Ovaj metod proverava da li je prosledjeni podatak niz. Vraća true ili false.
Podatak koji proveravamo se zadaje kao parametar.

Primer:

var niz7 = [45, 6, "Ivana", 88];
console.log(Array.isArray(niz7)); // vraća true

Još neki primeri:

console.log(Array.isArray("nešto")); // false
console.log(Array.isArray([])); // true
console.log(Array.isArray({})); // false
console.log(Array.isArray(33,44)); // false
console.log(Array.isArray(new Array(33,44))); // true
console.log(Array.isArray(true)) // false


.findIndex()

Ovaj metod proverava za svaki elemenat niza da li ispunjava uslov i vraća indeks prvog elementa koji ispunjava zadati uslov.Ukoliko ni jedan elemenat niza ne ispunjava uslov vraća -1.

Primer:

let niz8 = [19, 29, 39, 49, 59];
function nadjiPrviBroj(indeks) {
return indeks >= 40;
}
console.log(niz8.findIndex(nadjiPrviBroj)); // vraća 3
// broj 49 je prvi koji ispunjava uslov,on on je četvrti elemenat niza i ima indeks 3


.reduceRight()

Metod reduceRight() prolazi kroz niz od poslednjeg ka prvom elementu.

Primer:

var niz9 = [200, 400, 600, 800];
function oduzmi(ukupno, brojevi) {
return ukupno - brojevi;
}
console.log(niz9.reduceRight(oduzmi)); // kreće sa desna na levo i oduzima sve brojeve
